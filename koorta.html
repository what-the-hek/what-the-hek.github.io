Koorta

Brief

As a team of five, create a game using the theme "connection". this game was created during a 2 day game jam run by Level Her Up

I was responsible for programming the game using Unity and C#.

Some of the features for the game include:

  • random generated directions for left, right, up and down arrows
  • user keyboard input triggers specific animations for spider to make it dance
  • hidden counter to track the number of matching user actions to directions
  • messages to indicate a win or loss based on threshold (7 out of 10 dance correct moves)

You can view the Unity files and code on Github.

Koorta will be available on itch.io soon, stay tuned!
