Tech Diversity

Group project

The purpose of this website is to showcase diversity in tech and inspire women to pursue a tech career.

My main role in this group project was to create the 'hero' section and style the user profile cards.

I used JavaScript and React to connect the back end data (Django Rest Framework) with the front end. I deployed the front end on Netlify.

My contributions include:

  • a carousel to showcase the 'hero' cards
  • the ability to add, edit or hide hero cards when logged in
  • styling the hero and profile cards
