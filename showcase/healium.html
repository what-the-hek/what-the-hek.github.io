Healium

Solo crowdfunding project

Healium is a crowdfunding website designed to help people financially recover from family and domestic abuse.

I chose the theme for this website based on my 14 years experience working in emergency services. The soft design is intentional, to create a safe and comforting feeling for the user.

Key features include:

  • a 'quick exit' button that logs the user out and redirects the page, for their personal safety
  • randomly generated user names to help protect the users identity
  • the ability for users to create a crowdfunding project, or pledge to an existing project
