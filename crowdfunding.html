Brief

Create a corwdfunding website using Django Rest Framework (DRF) for the backend, and JavaScript React for frontend.

I decided to theme the website around people who've experienced family and domestic abuse. I chose this theme based on my background experience in emergency services.

Some of the features for this website include:

  • random generated usernames to retain anonymity and avoid accidental disclosure of name
  • quick-exit feature as a safety precaution
  • create a new fundraising project
  • pledge an amount to an existing project

You can view the full front-end code and back-end code on GitHub. Visit the Healium website, hosted on Netlify.
